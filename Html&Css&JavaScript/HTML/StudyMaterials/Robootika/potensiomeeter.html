Potensiomeeter

Kasutatud komponendid:

  • Arduino UNO plaat (1tk)
  • Arendusplaat (1tk)
  • Juhtmed (12tk)
  • Takisti (6tk, 220Ω)
  • LED punane (2tk)
  • LED kollane (2tk)
  • LED roheline (2tk)

Töö protsessid:

  • Ühendasin ledid värvide järgi.

  • 1. režiim - Küünlaeffekt
  • 2. režiim - Värvid järjest vilguvad
  • 3. režiim - Kõik värvid vilguvad
  • 4. režiim - Värvid lähevad üksteise järel põlema ja kustuvad üks haaval
  • 5. režiim - Värvid lähevad üksteise järel põlema ja kustuvad üks haaval alustades sellest mis viimasena põlema läks(videol ei ole näha, simulatsioonis on)
  • 6. režiim - Värvipaarid vilguvad üksteise järel
