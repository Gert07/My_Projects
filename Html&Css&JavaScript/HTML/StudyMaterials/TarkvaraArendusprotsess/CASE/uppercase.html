Mis on upper CASE vahend?

"Ülemise taseme" CASE-vahendeid toetavad peamiselt analüüsimist ja projekteerimist. Ennekõige leiavad nad kasutust visualiseerimisel, erinevate skeemide koostamisel ja samuti ka dokumentatsiooni genereerimiseks. Toetavad traditsiooniliste diagrammikeelte kasutamist (andmemudelid, UML-skeemid jne).

Peamised funktsioonid:

  • Nõuete modelleerimine
  • Protsesside modelleerimine
  • Andmemudelite loomine
  • Süsteemi arhitektuuri kirjeldamine
  • Dokumentatsiooni genereerimine
  • Projektihalduse toetamine
  • Kasutajaliidese prototüüpimine
